# Slava
Автор: Суслов Вячеслав Александрович
Описание программы:
Это приложение предназначено для отслеживания личных расходов. Вы можете добавлять расходы с указанием суммы, категории и даты. Возможна фильтрация по категории и дате, а также просмотр общей суммы за выбранный период. Все данные сохраняются в JSON-файл и загружаются при запуске.
Как использовать:

Введите сумму, выберите категорию и укажите дату, затем нажмите «Добавить расход».
Для фильтрации выберите категорию и/или дату и нажмите «Применить фильтр».
Чтобы сбросить фильтр, нажмите «Сбросить фильтр».
Все расходы сохраняются автоматически.

Примеры:

Добавьте расходы: 50 за еду, 200 за транспорт.
Фильтр по категории «Еда» и дате «2024-04-01».
Посмотрите сумму расходов за выбранный период.
